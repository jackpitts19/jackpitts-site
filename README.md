# Jack Pitts. Personal Site

Single-file personal website for Jack Pitts. Founder of Salt Creek Advisory and HelmIQ, host of The Making Of Hosted By Jack Pitts.

## Structure

- `index.html` is the whole site. Six tabbed chapters, inline CSS and JS, no build step, no dependencies.
- `assets/` holds the headshot and logos. Keep it next to index.html.

## Run locally

Open `index.html` in a browser. That's it.

## Deploy

Any static host works (GitHub Pages, Vercel, Netlify, Cloudflare Pages). For GitHub Pages: repo Settings, Pages, deploy from the main branch root.
